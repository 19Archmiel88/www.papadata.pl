# Glossary

- DEMO: tryb demo (UI = PROD, dane/akcje mockowane).
- PROD: tryb produkcyjny (pełne API, billing i uprawnienia).
- Trial: okres próbny z ograniczeniami i bannerami.
- Papa AI: drawer z AI chatem dostępny w dashboardzie.
- Guardian: widok alertów/ryzyk jakości danych.
- Pipeline: widok przepływów danych i ich statusów.
- KPI: kluczowe wskaźniki biznesowe.
- ROAS: return on ad spend.
- CPA: cost per acquisition.
- CVR: conversion rate.
- P&L: Profit & Loss (wynik finansowy).
- FTE: pełny etat (Full-Time Equivalent).
- Retencja: okres przechowywania danych w planie.
- Write-action: akcja zapisująca (eksport, alert, integracja).
- Read-only: tryb zablokowanych write-actions (brak aktywnej subskrypcji).
