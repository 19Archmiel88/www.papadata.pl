# Roadmapa

## Faza 0: Demo/POC (obecna)
- Landing + dashboard demo
- Mocki danych/integracji
- Integracja AI (stream/cancel/safety)
- Podstawy i18n + theme

## Faza 1: MVP (pierwsza wartość biznesowa)
- 2–3 realne integracje (np. GA4 + Meta Ads + Shopify)
- Persystencja konfiguracji klienta
- Minimalny model danych + katalog KPI
- Auth + podstawowy RBAC
- Export CSV/JSON (min. KPI + tabele)
- Observability (metryki, logi, alerty)

## Faza 2: Scale
- Kolejne integracje i “connectors”
- Automatyczne quality checks (freshness, completeness)
- Raporty shareable + role-based views
- FinOps: kontrola kosztów zapytań i AI

## Faza 3: Enterprise
- SSO/SAML (opcjonalnie)
- Audyt i compliance w standardzie enterprise
- SLA/Runbook dojrzałe
- Rozbudowany katalog metryk i governance
