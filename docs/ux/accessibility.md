# Dostępność (WCAG 2.1 AA)

## Cel
Aplikacja powinna być używalna bez myszy, z czytelnym kontrastem i przewidywalnym focusem.

## Checklist (minimum)
- pełna nawigacja klawiaturą (Tab/Shift+Tab/Enter/Escape)
- focus visible na wszystkich interakcjach
- sensowne etykiety aria dla ikon i przycisków bez tekstu
- modale/drawery: focus trap + powrót focusu
- kolory: kontrast tekstu i elementów UI
- wykresy: tooltipy dostępne z klawiatury, alternatywne opisy trendów
- reduced motion: ograniczenie animacji

## Deklaracja dostępności
Utrzymuj `docs/legal/accessibility-statement.md` oraz link w stopce.
